### `clone`

- 用于从远程仓库拷贝项目

  ```shell
  git clone git@github.com:zhiweiHong/font-end-note.git
  ```

### `add`

- 用于将文件添加到`.git`，进行版本管理

- `.`表示添加全部

  ```shell
  git add filename
  ```

### `commit`

- 将文件提交到本地仓库中

  ```shell
  git commit filename
  ```

### `push`

- 将文件提交到远程仓库中

  ```shell
  git push origin master
  ```

### `rm`

- 删除远程仓库文件

  ```shell
  git rm filename
  ```
